# Universities App

Каталог университетов в стиле Booking.com с функциями:
- Поиск и фильтрация университетов
- Разное поведение до/после регистрации
- 'Цель = Избранное' для пользователей
- Детальная информация с инструкциями 'Как поступить'
- Адаптивный дизайн с Tailwind CSS

## Технологии
- React + TypeScript
- Vite
- Tailwind CSS
- Framer Motion
- Lucide React
